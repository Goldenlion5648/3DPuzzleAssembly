# 3DPuzzleAssembly 
## By Colter Boudinot

A game where the player must try to fit together the pieces to make a cube.

Randomly generated, where you are allowed to enter a seed to try again or race your friends in.


Download it here: https://drive.google.com/open?id=1y6i9MLRH0baulbVEH1ScRQSo4egPoF5G

Or you can play it in the browser (generally slower, and works best in firefox): https://goldenlion5648.itch.io/3d-puzzle-assembly
